3.1 Basic Cable Types and Connectors

üåê Network Cables

Copper Cables

Cat 5/5e: Up to 100 Mbps (Cat 5), 1 Gbps (Cat 5e)

Cat 6/6a: Up to 10 Gbps, better shielding

Coaxial: Cable internet, older networks

STP vs UTP: Shielded vs Unshielded Twisted Pair

Fiber Optic

Single-mode: Long distance, laser light

Multi-mode: Shorter distance, LED light

Connectors: ST, SC, LC

Benefits: No EMI, longer distances, higher speeds

üîß Interactive: T568A vs T568B Wiring

T568A: Green-White, Green, Orange-White, Blue, Blue-White, Orange, Brown-White, Brown

T568B: Orange-White, Orange, Green-White, Blue, Blue-White, Green, Brown-White, Brown

Straight-through: Both ends same standard. Crossover: One A, one B.

üîå Peripheral & Video Cables

USB Standards

USB 2.0: 480 Mbps, Type-A/B, mini/micro

USB 3.0: 5 Gbps, blue connectors

USB-C: Reversible, up to 40 Gbps

Lightning: Apple proprietary

Video Connectors

HDMI: Digital audio/video, 4K support

DisplayPort: High resolution, daisy chaining

DVI: Digital Visual Interface

VGA: Legacy analog video

3.2 RAM Installation and Types

üíæ RAM Types and Characteristics

DDR Standards

DDR3: 1.5V, up to 2133 MHz

DDR4: 1.2V, up to 3200+ MHz

DDR5: 1.1V, up to 6400+ MHz

Each generation is NOT backward compatible

Form Factors

DIMM: Desktop memory (288 pins DDR4)

SODIMM: Laptop memory (260 pins DDR4)

ECC RAM: Error correction for servers

Registered: Buffered memory for stability

üîÑ Channel Configurations

Single Channel

One memory module or modules in same channel

Basic performance, no speed boost

Dual Channel

Two identical modules in paired slots

~10-15% performance increase

Most common configuration

Triple/Quad Channel

Three or four modules

Higher-end systems

Better memory bandwidth

üéØ Interactive: RAM Slot Configuration

Typical dual-channel setup: Install matching RAM in slots 1&3 or 2&4

Color coding: Matching colored slots = same channel

Best practice: Same brand, speed, and capacity for stability

3.3 Storage Devices

üíΩ Hard Drive Types

Traditional HDDs

5,400 RPM: Laptop drives, quiet, power efficient

7,200 RPM: Desktop standard, good performance

10,000 RPM: Enterprise, high performance

15,000 RPM: Enterprise, maximum speed

Solid State Drives

SATA SSD: 2.5" form factor, up to 6 Gbps

NVMe: PCIe interface, much faster

M.2: Compact form factor for laptops

mSATA: Legacy compact form factor

üîÑ RAID Configurations

RAID 0 - Striping

Drives: 2 minimum

Benefit: Increased performance

Risk: No redundancy - any drive fails, all data lost

RAID 1 - Mirroring

Drives: 2 drives

Benefit: Complete redundancy

Cost: 50% storage capacity

RAID 5 - Striping with Parity

Drives: 3 minimum

Benefit: Performance + redundancy

Capacity: n-1 drives usable

RAID 10 - Mirror + Stripe

Drives: 4 minimum

Benefit: Best performance + redundancy

Cost: 50% storage capacity

3.5 Power Supply Installation and Configuration

‚ö° Power Supply Specifications

Input Voltage

110-120 VAC: North America standard

220-240 VAC: International standard

Auto-switching: Detects input voltage

Output Voltages

+3.3V: Modern components, RAM

+5V: USB ports, older components

+12V: CPU, graphics cards, fans

-12V: Legacy components (rare)

üîå Power Connectors

24-pin: Main motherboard power

4/8-pin CPU: Additional CPU power

20-pin adapter: Legacy compatibility

Component Power

SATA power: Modern drives

Molex: Legacy drives, fans

PCIe 6/8-pin: Graphics cards

üìä Interactive: Wattage Calculator

üîß Power Supply Types

Benefit: No single point of failure
