3.1 - Troubleshooting Windows OS Issues

💀 Blue Screen of Death (BSOD)

Critical system errors that cause Windows to stop and restart

Troubleshooting BSOD:

  • Check Event Viewer for error codes
  • Update or rollback device drivers
  • Run memory diagnostic (mdsched.exe)
  • Check for hardware issues
  • Boot in Safe Mode to isolate issues

🐌 Degraded Performance

System running slower than expected

Improving Performance:

  • Check Task Manager for resource usage
  • Disable unnecessary startup programs
  • Run Disk Cleanup (cleanmgr.exe)
  • Defragment traditional hard drives
  • Add more RAM if needed
  • Check for malware

🚫 Boot Issues

System fails to start properly

Boot Troubleshooting:

  • Use Windows Recovery Environment
  • Run Startup Repair
  • Check Boot Configuration Data (BCD)
  • Use System File Checker (sfc /scannow)
  • Verify hardware connections

⚙️ Services Not Starting

Essential system services failing to load

Service Troubleshooting:

  • Use services.msc to check service status
  • Check service dependencies
  • Review system logs for errors
  • Restart services manually
  • Check user account permissions

💡 Pro Tip:

Always check Event Viewer first when troubleshooting Windows issues. It provides detailed error information that can guide your troubleshooting efforts.

3.2 & 3.3 - Mobile OS and Application Issues

Common Mobile Issues

📱 Application Fails to Launch

Apps won't start or crash immediately

Solutions:

  • Force close and restart the app
  • Clear app cache and data
  • Update the application
  • Restart the device
  • Reinstall the app if necessary

🔋 Battery Life Issues

Device battery drains quickly

Battery Optimization:

  • Check battery usage statistics
  • Reduce screen brightness
  • Disable unnecessary background apps
  • Turn off location services when not needed
  • Update apps and OS
  • Enable power saving mode

📡 Connectivity Issues

Wi-Fi, Bluetooth, or cellular problems

Connectivity Fixes:

  • Wi-Fi: Forget and reconnect to network
  • Bluetooth: Clear Bluetooth cache
  • Cellular: Check APN settings
  • Toggle airplane mode on/off
  • Reset network settings
  • Update carrier settings

🔄 Screen Rotation Issues

Screen doesn't rotate automatically

Screen Rotation Fixes:

  • Check rotation lock setting
  • Calibrate accelerometer
  • Restart the device
  • Clear system cache
  • Update device drivers (Android)

⚠️ Security Considerations

Mobile security issues include unauthorized app installations, root access/jailbreaking, and malicious applications. Always verify app sources and keep devices updated.

3.4 - PC Security Issues

Common Security Symptoms

🌐 Unable to Access Network

Complete loss of network connectivity

Network Troubleshooting:

  • Check physical connections
  • Verify IP configuration (ipconfig)
  • Test with different devices
  • Check for malware blocking connections
  • Reset network adapter

⚠️ False Security Alerts

Fake antivirus or system warnings

Dealing with Scareware:

  • Don't click on suspicious alerts
  • Close browser/application immediately
  • Run legitimate antivirus scan
  • Check for unwanted browser extensions
  • Clear browser cache and cookies

🌐 Browser Problems

Pop-ups, redirections, slow performance

Browser Security Fixes:

  • Remove suspicious extensions
  • Reset browser settings
  • Clear browsing data
  • Update browser to latest version
  • Check proxy settings
  • Scan for adware/malware

🛡️ Prevention Tips:

  • Keep OS and software updated
  • Use reputable antivirus software
  • Enable Windows Defender Firewall
  • Be cautious with email attachments
  • Use strong, unique passwords

Malware Removal Process

CompTIA 10-Step Malware Removal Process

Investigate and Verify Malware Symptoms

Talk to the user, identify symptoms, and confirm malware presence through system behavior analysis.

Quarantine Infected System

Disconnect from network to prevent spread and protect other systems from infection.

Disable System Restore (Windows)

Prevent malware from hiding in restore points that could reinfect the system later.

Remediate Infected Systems

Use updated antivirus/anti-malware tools to scan and remove threats. Boot from external media if necessary.

Update Anti-malware Software

Ensure you have the latest virus definitions and security updates before scanning.

Scan and Remove Malware

Perform thorough scans in safe mode or preinstallation environment for stubborn infections.

Reimage/Reinstall if Necessary

For severe infections, complete OS reinstallation may be the most effective solution.

Schedule Scans and Updates

Set up automatic scanning schedules and ensure regular security updates.

Enable System Restore

Re-enable System Restore and create a clean restore point for future recovery.

Educate the End User

Provide training on safe computing practices to prevent future infections.
