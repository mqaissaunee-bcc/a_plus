Key Concept: TCP vs UDP

TCP (Transmission Control Protocol): Connection-oriented, reliable, slower
UDP (User Datagram Protocol): Connectionless, faster, less reliable

Protocol Comparison

Protocol Type Purpose Example Use
HTTPS TCP Secure web browsing Online banking
DHCP UDP Automatic IP assignment Home router configuration
